<h3 align="center">Relative Magnitudes Analysis</h3>

Author: Sydney Gable, University of Michigan | gablesyd@umich.edu

<h5 align="left">Overview:</h3>
The relative magnitude analysis codes provided here allow the user to generate calibrated earthquake magnitudes 
using the relative magnitude method. Further information on the methodology and recommended user approaches can be found in:<br>
Gable, S.L., and Y. Huang (2025) Quantifying Magnitude Uncertainty of the 2019 Ridgecrest Earthquake Sequence Through a Sensitivity 
Study of the Relative Magnitude Method. Bull. Seismol. Soc. Am., 115 (3)

<h5 align="left">User Information:</h3>
A user manual and further documented codes are coming very soon! Please note that this page is continuously being updated. If you plan to use the Relative Magnitude Analysis codes for your own work, please contact gablesyd@umich.edu for the most up-to-date information.
